JavaScript - Fundamentos

Resumo

JavaScript é uma linguagem de programação de alto nível, interpretada e com tipagem dinâmica. É uma das principais tecnologias da web, juntamente com HTML e CSS.

Tipagem Dinâmica

JavaScript é uma linguagem de tipagem dinâmica, ou seja, não é necessário declarar o tipo de uma variável, o tipo é definido automaticamente conforme o valor atribuído.

Tipos de Variáveis

  • Number: qualquer número (inteiro ou decimal);
  • String: sequência de caracteres (textos);
  • Boolean: valores lógicos (true ou false);
  • Array: coleção ordenada de valores (lista);
  • Object: coleção de pares chave:valor;
  • Null: valor nulo proposital;
  • Undefined: variável declarada mas sem valor atribuído.

Declaração e Atribuição

  • Declaração: reservar um espaço na memória. let nome;
  • Atribuição: dar um valor a essa variável. nome = "João";
  • Podemos fazer os dois juntos: let nome = "João";

Regras para Variáveis

  • Não podem conter espaços ou caracteres especiais;
  • Não podem ter o mesmo nome de palavras reservadas do JavaScript;
  • JavaScript é case-sensitive (diferencia letras maiúsculas e minúsculas).

Palavras-chave

  • let: cria uma variável com escopo de bloco;
  • const: cria uma constante (valor não muda);
  • var: mais antiga, com escopo de função (menos recomendada hoje).

Operadores de Atribuição

Os operadores de atribuição em JavaScript são usados para atribuir valores a variáveis.

  • = : atribuição simples;
  • += : adição e atribuição;
  • -= : subtração e atribuição;
  • *= : multiplicação e atribuição;
  • /= : divisão e atribuição;
  • %= : módulo e atribuição;
  • **= : exponenciação e atribuição (ex: x **= 2 equivale a x = x ** 2).

Operadores Aritméticos

  • + → adição
  • - → subtração
  • * → multiplicação
  • / → divisão
  • % → resto da divisão
  • ** → exponenciação
  • ++ → incremento (aumenta 1)
  • -- → decremento (diminui 1)

Conversão de Tipos

  • Number() → converte strings em números;
  • parseFloat() → converte para número com casas decimais;
  • parseInt() → converte para número inteiro;
  • toString() → transforma número em string;
  • toFixed(n) → define número de casas decimais;
  • toPrecision(n) → define quantidade de dígitos significativos.

Arrays

Arrays são listas que armazenam vários valores em uma única variável.

const frutas = ["Banana", "Maçã", "Laranja"];
console.log(frutas[0]); // Banana
  • Os índices começam do zero;
  • Podemos acessar e modificar os itens pelo índice.

Métodos úteis:

  • toString() → converte o array para string;
  • length → retorna a quantidade de elementos;
  • push() → adiciona item ao final;
  • pop() → remove o último item.

Objetos

Objetos armazenam dados em pares chave: valor.

const pessoa = {
    nome: "Maria",
    idade: 30,
    profissao: "Programadora"
};
console.log(pessoa.nome); // Maria
  • Podemos acessar usando obj.chave ou obj["chave"].

Estrutura de Dados: Vetores

Vetores são estruturas de dados lineares que armazenam uma sequência de elementos do mesmo tipo, acessados por índices numéricos.

No JavaScript, os vetores são representados pelos Arrays.

Por que usar vetores?

  • Organização e agrupamento de dados.
  • Facilidade de acesso com base em índice.
  • Boa performance em operações simples.

Exemplo de uso de vetor:

const numeros = [10, 20, 30, 40];
console.log(numeros[2]); // 30
